Face book script:

this script basiclly sends a message to all the list available under your birthday friends.

requirement:
1- python 3 installed
2- the script was made for firefox but if you want to use any other browser just change the webdriver
3- provide your username and password in the email field and password and let the scipt do the rest
4- run the script >> python3 app.py 
5- in case you entered the wrong password you will be prompted in the command line to enter the right one.

the app has a lot of room for improvements i still have some ideas in mind will work on them, but if you have any suggestions please let me know.
